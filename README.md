50.### 文字被些字符包围   
51.> 文字被些字符包围   
52.>   
53.> 只要再文字前面加上>空格即可   
54.>   
55.> 如果你要换行的话,新起一行,输入>空格即可,后面不接文字   
56.> 但> 只能放在行首才有效   
57.  
58.### 文字被些字符包围,多重包围   
59.> 文字被些字符包围开始   
60.>   
61.> > 只要再文字前面加上>空格即可   
62.>   
63.>  > > 如果你要换行的话,新起一行,输入>空格即可,后面不接文字   
64.>   
65.> > > > 但> 只能放在行首才有效   
66.  
67.### 特殊字符处理   
68.有一些特殊字符如<,#等,只要在特殊字符前面加上转义字符\即可<br />   
69.你想换行的话其实可以直接用html标签\<br /\>
